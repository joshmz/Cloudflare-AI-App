# Cloudflare-AI-App
Artificial Inteligence app for cloudflare 2026 internship submission

Uses Ollama as the local AI solution with llama3:8b as the model.

Create AI chatbot which takes user input via chat.

Has memory of current conversation and retains memory from previous conversations.
Uses memory as context for future conversations.

Possible external technologies:
- Workflow / coordination (recommend using Workflows, Workers or Durable Objects)
- User input via chat or voice (recommend using Pages or Realtime)
